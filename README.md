# 概览
模型服务平台UModelVerse旨在为客户快速搭建AGI应用的能力，作为AGI的应用者，客户无需关注底层的算力资源调度、基础环境的模型部署，仅需要通过UModelVerse控制台或者统一对外API，即可快速搭建专属AGI应用。
