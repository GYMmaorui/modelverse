# **计费说明**



| **功能模块** | **计费方式** | **计费说明**                                                 |
| ------------ | ------------ | ------------------------------------------------------------ |
| 模型微调     | 按量后付费   | 根据训练任务的实际运行时长进行计算。按小时收取训练算力费用，不足1小时按分钟计算。 |
| 服务管理     | 预付费       | 根据模型自动计算算力单元，用户指定副本数合计费用=单价 * 单副本算力单元数 * 副本数 |
| 对象存储     | 按量后付费       | 按数据存储量计费，具体详见[UCloud文档中心](https://docs.ucloud.cn/ufile/bill/new) |
| API调用     | 按量后付费       | 根据调用token计费 |
